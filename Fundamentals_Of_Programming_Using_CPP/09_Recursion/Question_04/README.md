# Sum of the Array Elements

Write a C++ program to find the sum of array elements using recursion.

### Sample Input 1:

```
5
1
2
3
4
5
```

### Sample Output 1:

```
15
```

### Sample Input 2:

```
2
1 2
```

### Sample Output 2:

```
3
```

### Sample Input 3:

```
2
9 6
```

### Sample Output 3:

```
15
```

### Sample Input 4:

```
5
10 20 30 40 50
```

### Sample Output 4:

```
150
```
