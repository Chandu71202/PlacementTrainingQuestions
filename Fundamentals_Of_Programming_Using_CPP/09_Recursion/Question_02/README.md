# Prime Number

Write a C++ program to find whether the given number is a prime number or not using recursion.

### Sample Input 1:

```
13
```

### Sample Output 1:

```
Prime Number
```

### Sample Input 2:

```
199
```

### Sample Output 2:

```
Prime Number
```

### Sample Input 3:

```
246
```

### Sample Output 3:

```
Not a Prime Number
```

### Sample Input 4:

```
101
```

### Sample Output 4:

```
Prime Number
```
