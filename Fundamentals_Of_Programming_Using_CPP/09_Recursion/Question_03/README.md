# Decimal to Binary

Write a C++ program to convert a decimal number to a binary number by recursion.

### Sample Input 1:

```
10
```

### Sample Output 1:

```
1010
```

### Sample Input 2:

```
123
```

### Sample Output 2:

```
1111011
```

### Sample Input 3:

```
3
```

### Sample Output 3:

```
11
```

### Sample Input 4:

```
444
```

### Sample Output 4:

```
110111100
```
