# String-count the vowels

Write a C++ program to count the number of vowels in the given string.

### INPUT & OUTPUT FORMAT:

Input consists of 1 string. Output print the number of Vowels.

### Sample Input 1:

```
face
```

### Sample Output 1:

```
Number of vowels: 2
```

### Sample Input 2:

```
uppercase
```

### Sample Output 2:

```
Number of vowels: 4
```

### Sample Input 3:

```
APPLE
```

### Sample Output 3:

```
Number of vowels: 2
```

### Sample Input 4:

```
freshgrad
```

### Sample Output 4:

```
Number of vowels: 2
```
