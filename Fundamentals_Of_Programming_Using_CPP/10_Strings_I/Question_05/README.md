# Print the String

Write a C++ program to scan a string from the user and to print it.

### INPUT FORMAT: 

Input consists of 1 string. 

### Sample Input 1:

```
Face
```

### Sample Output 1:

```
The string is Face
```

### Sample Input 2:

```
String
```

### Sample Output 2:

```
The string is String
```

### Sample Input 3:

```
FRESHGRAD
```

### Sample Output 3:

```
The string is FRESHGRAD
```

### Sample Input 4:

```
Length
```

### Sample Output 4:

```
The string is Length
```
