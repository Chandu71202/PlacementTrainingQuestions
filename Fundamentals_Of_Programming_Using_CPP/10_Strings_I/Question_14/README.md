# String-Delete vowels

Write a program to delete the vowels in the given string and display the string without vowels.

### Sample Input 1:

```
face
```

### Sample Output 1:

```
fc
```

### Sample Input 2:

```
FACENOW
```

### Sample Output 2:

```
FCNW
```

### Sample Input 3:

```
CodeLAB
```

### Sample Output 3:

```
CdLB
```

### Sample Input 4:

```
Words
```

### Sample Output 4:

```
Wrds
```
