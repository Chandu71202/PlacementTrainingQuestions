# Remove all Characters in Second String which are present in First String

Write a program to remove all characters in second string which are present in first string.

### Sample Input 1:

```
Motor
Motorcycle
```

### Sample Output 1:

```
cycle
```

### Sample Input 2:

```
Bike
BikeWay
```

### Sample Output 2:

```
Way
```

### Sample Input 3:

```
Sample
SampleIn
```

### Sample Output 3:

```
In
```
