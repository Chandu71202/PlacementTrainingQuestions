# Print the string - II

Write a C++ program to scan the string from the user and to print it.

### INPUT FORMAT:

Input consists of 1 string.

### Sample Input 1:

```
Hello World
```

### Sample Output 1:

```
The string is Hello World
```

### Sample Input 2:

```
Welcome to my Home
```

### Sample Output 2:

```
The string is Welcome to my Home
```

### Sample Input 3:

```
Well done
```

### Sample Output 3:

```
The string is Well done
```
