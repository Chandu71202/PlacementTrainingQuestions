# Count the sum of numbers in the alphabet

Given a string containing alphanumeric characters, calculate the sum of all numbers present in the string.

### Sample Input 1:

```
1abc23
```

### Sample Output 1:

```
24
```

### Sample Input 2:

```
Fa33ce1
```

### Sample Output 2:

```
34
```

### Sample Input 3:

```
fresh11grad22
```

### Sample Output 3:

```
33
```

### Sample Input 4:

```
Code22lab1
```

### Sample Output 4:

```
23
```
