# String-Alphabet Frequency

Write a C++ program to count the frequency of alphabets in a given string.

### INPUT FORMAT:

Input consists of 1 string.

### Sample Input 1:

```
google
```

### Sample Output 1:

```
e 1
g 2
l 1
o 2
```

### Sample Input 2:

```
chrome
```

### Sample Output 2:

```
c 1
e 1
h 1
m 1
o 1
r 1
```

### Sample Input 3:

```
Codelab
```

### Sample Output 3:

```
C 1
a 1
b 1
d 1
e 1
l 1
o 1
```

### Sample Input 4:

```
Freshgrad
```

### Sample Output 4:

```
F 1
a 1
d 1
e 1
g 1
h 1
r 2
s 1
```
