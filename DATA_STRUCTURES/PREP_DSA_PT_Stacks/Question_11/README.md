# Question 11

Write a program to insert elements in the stack and display the topmost element.

### Input Format

Number of inputs<br>
Stack elements

### Output Format

Topmost element in the stack.

## Sample Testcases

### Input 1

```
5
3 9 4 7 6
```

### Output 1

```
6
```

## Whitelist

- Set 1 : `push`
- Set 2 : `peek`
- Set 3 : `top`

## Blacklist

- `[`
- `]`
