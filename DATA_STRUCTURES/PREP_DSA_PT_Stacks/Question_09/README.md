# Question 09

Write a program to reverse a string **s** using stack.

### Input Format

Your editor reads a single line containing string called **s**.

### Output Format

Displays the reversed string.

## Code Constraints

**1 <= s <= 100**

## Sample Testcases

### Input 1

```
kumar
```

### Output 1

```
ramuk
```

### Input 2

```
Sample Input
```

### Output 2

```
tupnI elpmaS
```

## Whitelist

- Set 1 : `pop`
- Set 2 : `push`

## Blacklist

- `strrev`
