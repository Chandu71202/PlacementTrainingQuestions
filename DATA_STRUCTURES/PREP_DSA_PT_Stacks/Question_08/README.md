# Question 08

Write a program to delete the top element in a stack and display the stack elements.

### Input Format

Number of inputs <br>
Stack Elements

### Output Format

Deleted element in the stack <br>
Stack elements <br><br>
Refer sample output.

## Sample Testcases

### Input 1

```
5
5 6 4 9 8
```

### Output 1

```
Deleted element is 8
The elements in the stack
9
4
6
5
```
