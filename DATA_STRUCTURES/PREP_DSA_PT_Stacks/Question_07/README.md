# Question 07

Write a program to convert an infix expression to a post-fix expression using Stack concept.

### Input Format

Input consists of an infix expression.

### Input Format

Output displays the post-fix expression of the input.

## Sample Testcases

### Input 1

```
(a+b)*(a-b)
```

### Output 1

```
ab+ab-*
```

### Input 2

```
A+(B*C-(D/E^F)*G)*H
```

### Output 2

```
ABC*DEF^/G*-H*+
```
