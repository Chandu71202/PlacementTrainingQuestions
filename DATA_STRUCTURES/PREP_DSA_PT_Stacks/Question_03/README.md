# Question 03

Write a C program to implement a stack and search for an element in the stack.

### Input Format

Number of inputs <br>
Stack elements <br>
Element to be searched in the stack 

### Output Format

Output prints whether the element is found or not.

## Sample Testcases

### Input 1

```
5
1 2 3 4 5
3
```

### Output 1

```
Element found
```

### Input 2

```
5
1 2 3 4 5
37
```

### Output 2

```
Element not found
```

## Whitelist

- Set 1 : `push()`
- Set 2 : `display()`
- Set 3 : `top`

## Blacklist

- `[`
- `]`
