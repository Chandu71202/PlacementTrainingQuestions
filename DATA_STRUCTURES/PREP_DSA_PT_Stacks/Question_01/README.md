# Question 01

Write a C program to sort the set of elements in descending order using stack.

### Input Format

Number of elements. <br>
Input the stack elements.

### Output Format

Elements in descending order.

## Sample Testcases

### Input 1

```
5
8 10 4 2 7
```

### Output 1

```
10 8 7 4 2
```

## Whitelist

- Set 1 : `top`
- Set 2 : `push`
- Set 3 : `pop`

## Blacklist

- `[`
- `]`
